# NMAP AND SCAPY DOCUMENTATION

## 1. Objective
This project is the documentation of Nmap and Scapy practical exercises.

## 2. Goal
To perform Host discovery, Operating system detection, SMB scanning, packet sniffing, Create and Send ICMP packets within an internal lab network.
The exercises demonstrate skills in reconnaissance, enumeration, and packet analysis.

## 3. Environment
   -  Operating System: Kali Linux
   - Tools: Nmap, Scapy, tcpdump
   - Network: 10.6.6.0/24
   - Target Host: 10.6.6.23
